What is PyWeb ?
PyWeb is a very powerful Web library coded in the Python Programming Language.

What are these features ?
Features are:

  • Easy to learn if you already know Python
  • Doesn't need a Python web server: all the code is transformed into pure xHTML/PHP code
  • Support PHP throught PyWebDynamic module.
  • PyWebDynamic uses SQLite so it has very high performance.
