Download

You can download PyWeb from its GitHub repository as a ZIP archive or a TAR archive.

The archive contains: the library in lib/, the source in src/, the LICENSE and the README.

PyWebDynamic is included. To use it, you must have a Web HTTP server supporting PHP (almost all the Web servers support PHP).
